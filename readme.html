1- Descargar Plugin 2- Extraer archivo prueba_wordpres-main 3- Dirigirse a donde wp-content/plugins => testPlugin (plugin creado) 4- Dirigirse nuevamente a la carpeta dewp-content/themes => arrastrar twentytwentyone (tema con el widget creado al final ) 5- En el panel de WordPress ir a Plugins Test Plugin => Activar 6- Una vez activado en el dashboard o menú aparecerá dos elementos nuevos: • Prueba/Configuración: En esta página, permite al usuario configurar algunas opciones específicas del plugin (por ejemplo, colores, texto). • Data Api: Esta pagina esta encargada de interactuar con el api de json Placeholder que es una api ficticia donde renderizamos algunos datos, y además agregamos una imagen simulando productos, utilice css grid y algunos estilos extra (utilice la etiqueta style, pero la forma óptima seria crear una carpeta con el nombre style/estilos.css y asi mantener correctamente el plugin) 7- El plugin contiene Integracion con woocomerce: diríjase a productos/editar/avanzado/ Número de Serie(campo personalizado) => podemos agregar un numero de serie y que se renderice y se guarde correctamente. 8- Utilizar Shortcode en Páginas/Entradas: Edita una página o entrada y utiliza el shortcode [info] para mostrar información específica de los productos. 9- Widget : Ir a Apariencia/Widgets, tocar “+” escribir product_info_widget => escribir un titulo y te trae informacion de los productos (limitados 3)
